О Проекте

VulnAnalysis — это обучающий веб‑сервис для сбора, хранения и визуализации уязвимостей из Национальной Базы Данных Уязвимостей (NVD, США). Мы автоматизируем:

  • Скачивание свежих CVE‑записей через NVD API.
  • Парсинг и сохранение ключевых метрик (CVSS, даты, CWE, ссылки).
  • Обновление базы данных по расписанию.
  • Интерактивные графики и дашборды для анализа трендов.

Цель проекта — продемонстрировать навыки Data Wrangling, PostgreSQL, Flask и D3.js, а также дать пользователям мощный инструмент для визуального анализа безопасности ПО.
